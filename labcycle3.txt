<h1> CSS selectors </h1>
<p align="center"> CSS selectors are used to find or select the HTML elements you want to style </p>
<h2> CSS Element Selector </h2>
<p> Element selectors select the HTML elements based on the element name </p>
<h2> CSS ID Selector </h2>
<p id="para1" > The id selector uses the id attribute of an HTML element to select a specific element 
<br>
To select an element with a specific id,write a (#) character,followed by the id of the element
</p>
<b> <u> Note:</u> An id name can not start with a number! </b>
<h2> Class Selector </h2>
<p class="para2"> Class selector slects the HTML elements with a specific class attribute <br>
To select an element with a specific class,write a (.) character,followed by the id of the element
</p>
<h2> Universal Selector </h2> The Universal Selector(*) selects all HTML elements
<h2> CSS Grouping Selector </h2> <b> The Grouping Selector selects all HTML elementswith same style definitions
<br> It will be better to group the selectors,to minimize the code
<br> To group selectors,separate selectors with a comma </b>
